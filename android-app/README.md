# 社媒 claw / Android App

这是一个用于社媒内容采集和浏览的 Android 客户端。

## 目录
- `app/`：Android 应用源码
- `app/src/main/java/`：主业务代码
- `app/src/main/res/`：资源文件

## 发布前注意
- 不要提交真实 API key、token、私钥、`local.properties`
- 服务器地址、账号等敏感默认值请放到本地配置或运行时输入
- 构建产物不要提交到仓库

## 构建
```bash
./gradlew assembleDebug
```

- 当前版本：`1.10`
